# 3. 교차 학습 경로

## 기초 경로

보건의료정보관리 개론 → 의료기록의 이해 → 의학용어 → 기록 실무

## 분류 경로

의학용어 → 의료기록 해석 → KCD/ICD → 주진단 선정 → 통합 코딩 사례

## 통계 경로

통계 기초 → 환자센서스 → 병원통계 공식 → 계산 문제 → 사례형 문제

## 디지털 경로

EMR/EHR → 의료정보 표준 → 데이터 품질 → 개인정보 보호 → AI와 데이터 거버넌스

## 실무 경로

병원 조직 → 의무기록 작성 → 기록 관리 → 완결성 분석 → 정보 제공 → 감사
