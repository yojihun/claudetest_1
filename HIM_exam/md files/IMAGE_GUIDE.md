# 1. 목적

이미지는 장식이 아니라 개념 이해와 회상을 돕는 학습 요소이다. 텍스트를 그대로 그림 안에 옮겨 적지 않는다.

# 2. 우선 제작할 시각자료

1. 흐름도
2. 비교표
3. 정보 구조도
4. 타임라인
5. 병원 업무 프로세스
6. 시스템 관계도
7. 통계 공식 해석도
8. 기록 분석 예시
9. 암등록·분류 의사결정 트리
10. 실무 상황 일러스트

# 3. 시각자료 선택 기준

| 학습 목적 | 권장 형식 |
|---|---|
| 순서 이해 | 흐름도 |
| 유사개념 구분 | 비교표 |
| 시스템 관계 | 아키텍처 다이어그램 |
| 역사적 발전 | 타임라인 |
| 계산식 이해 | 분자·분모 도식 |
| 사례 판단 | 의사결정 트리 |
| 업무 역할 | 스윔레인 |
| 기억 강화 | 단순 아이콘 또는 일러스트 |

# 4. 디자인 원칙

- 한 이미지에 하나의 핵심 아이디어를 담는다.
- 텍스트는 최소화한다.
- 선, 화살표, 그룹을 사용해 관계를 표현한다.
- 앱의 작은 화면에서도 읽을 수 있게 한다.
- 인쇄 시 흑백에서도 의미가 유지되어야 한다.
- 색상만으로 상태를 구분하지 않는다.
- 약어에는 필요할 때 전체 명칭을 함께 제공한다.
- 이미지 아래에 핵심 캡션을 둔다.

# 5. 앱용 기술 기준

- 흐름도와 아이콘은 SVG 우선
- 복잡한 일러스트는 PNG 또는 WebP
- 인쇄용 원본은 충분한 해상도 확보
- 파일명은 Asset ID와 일치
- 모든 이미지에 대체 텍스트 제공
- 투명 배경은 재사용성이 있을 때 사용
- 다크모드에서 보이지 않는 요소가 없는지 확인

# 6. 권장 비율

- 개념 카드: 4:3
- 넓은 프로세스: 16:9
- 양면 또는 긴 흐름: 16:8
- 모바일 세로 카드: 4:5 또는 9:16
- 아이콘: 1:1

비율보다 정보의 가독성을 우선한다.

# 9. 인물·사례 이미지

- 환자를 특정 인종·성별·연령의 고정관념으로 표현하지 않는다.
- 실제 환자를 재현하지 않는다.
- 병원 업무 역할을 명확하게 표현한다.
- 과도하게 공포스럽거나 자극적인 의료 장면을 피한다.
- 교육 대상이 성인 수험생임을 고려해 지나치게 유아적인 스타일을 피한다.

# 10. 이미지 검토 체크리스트

- [ ] 핵심 메시지가 3초 내에 이해되는가
- [ ] 본문을 중복하지 않는가
- [ ] 모바일에서 읽히는가
- [ ] 흑백에서도 관계가 유지되는가
- [ ] 대체 텍스트가 충분한가
- [ ] 화살표 방향이 정확한가
- [ ] 의학적·업무적 오류가 없는가
- [ ] 출처와 라이선스가 기록되었는가
