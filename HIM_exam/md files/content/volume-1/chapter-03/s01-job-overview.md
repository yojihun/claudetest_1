# 직무 개요

## 학습목표

1. 보건의료정보관리사의 13개 세부 직무 영역을 나열할 수 있다.
2. 각 직무 영역이 서로 어떻게 연결되는지 설명할 수 있다.

## 한 문장 정의

보건의료정보관리사의 직무는 기록관리, 분류, 통계, 암등록, 정보제공, 개인정보 보호, 데이터 품질관리, 연구지원, 표준화, 데이터 거버넌스, 임상문서개선, 부서 간 협업, 직업윤리라는 13개 세부 영역으로 구성된다.

## 왜 중요한가

Chapter 1에서 다룬 "정보 업무와 진료행위의 경계"라는 큰 틀을, Chapter 3에서는 각 세부 직무별로 구체화한다. 국가시험은 이 세부 직무 중 하나를 콕 집어 사례로 묻는 경우가 많다.

## 상세 설명

이 13개 영역은 독립적으로 존재하지 않는다. 예를 들어 분류(3) 업무의 정확성은 데이터 품질관리(7)와 연결되고, 데이터 품질관리는 다시 연구지원(8)의 기초 자료 신뢰성과 연결된다. 정보 제공(5)과 개인정보 보호(6)는 서로 긴장 관계에 있으며(제공 요청에 응하면서도 최소권한을 지켜야 함), 이 균형을 잡는 것이 실무의 핵심 난이도다.

## 비교표 또는 흐름

| 직무 군 | 포함 영역 |
|---|---|
| 정보 생산·가공 | 기록관리, 분류, 통계, 암등록 |
| 정보 유통·보호 | 정보 제공, 개인정보 보호 |
| 품질·표준 관리 | 데이터 품질관리, 표준화 |
| 조직·관리 | 데이터 거버넌스, 임상문서개선, 부서 간 협업 |
| 지원·원칙 | 연구 지원, 직업윤리와 책임 |

## 실무 사례

한 보건의료정보관리사가 연구자로부터 특정 질환 환자 데이터를 요청받았을 때, 정보 제공(연구지원) 업무와 개인정보 보호 업무가 동시에 작동해야 한다. 요청에 응하되 비식별 처리와 최소 필요 범위 제공 원칙을 함께 적용하는 것이 이 직무의 전형적인 실무 모습이다.

## 국가시험 포인트

- 특정 업무 사례가 13개 영역 중 어디에 해당하는지 판단하는 문제
- 두 영역이 동시에 관련된 사례(예: 정보 제공+개인정보 보호)를 통합적으로 판단하는 문제

## 자주 하는 실수

- 직무를 "기록관리" 하나로만 좁게 이해하는 것
- 영역 간 연결(예: 분류 정확성과 데이터 품질관리)을 놓치고 각 영역을 완전히 독립적으로 이해하는 것

## 관련 법령·표준

각 세부 직무의 관련 법령은 해당 Topic에서 개별적으로 다룬다.

AI 도구는 이 13개 영역 대부분에서 보조 도구로 활용될 수 있으나(분류 코드 추천, 이상 탐지 등), 각 영역의 최종 판단과 책임은 보건의료정보관리사에게 있다.

## 핵심 요약

- 보건의료정보관리사의 직무는 13개 세부 영역으로 구성되며 서로 연결되어 있다.
- 정보 제공과 개인정보 보호처럼 서로 긴장 관계에 있는 영역의 균형을 잡는 것이 실무의 핵심이다.
