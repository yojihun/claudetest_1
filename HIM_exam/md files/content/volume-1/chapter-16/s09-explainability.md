# 설명가능성

## 학습목표

1. 의료 AI에서 설명가능성(explainability)이 왜 중요한지 설명할 수 있다.
2. '블랙박스(black box)' 모델의 한계를 설명할 수 있다.

## 한 문장 정의

설명가능성은 AI 모델이 특정 예측이나 판단을 내린 근거를 사람이 이해할 수 있는 형태로 제시할 수 있는 능력을 말한다.

## 왜 중요한가

의료진이 AI의 예측을 신뢰하고 임상적으로 활용하려면, 왜 그런 예측이 나왔는지 이해할 수 있어야 하며, 설명 없이는 [편향과 공정성](s08-bias-and-fairness.md) 문제도 발견하기 어렵다.

## 선수 개념

- [편향과 공정성](s08-bias-and-fairness.md)

## 핵심 개념

- 설명가능성(Explainability): 모델의 판단 근거를 사람이 이해할 수 있게 제시하는 능력
- 블랙박스(Black Box) 모델: 내부 작동 방식을 사람이 직관적으로 이해하기 어려운 복잡한 모델
- 근거 시각화: 모델이 예측 시 어떤 입력 요소를 중요하게 반영했는지 보여주는 방법

## 상세 설명

복잡한 딥러닝 기반 모델은 성능이 높지만 내부 작동 방식이 사람이 이해하기 어려운 '블랙박스'인 경우가 많다. 설명가능성 기법은 모델이 예측 시 어떤 입력 요소(예: 특정 검사 수치, 영상의 특정 부위)를 중요하게 반영했는지 시각화해 보여준다. 이는 [편향과 공정성](s08-bias-and-fairness.md)에서 다룬 문제를 발견하는 데도 도움이 된다. 예를 들어 모델이 진단과 무관한 요소(예: 촬영 장비 종류)를 근거로 판단하고 있다면, 이는 편향의 신호일 수 있다.

## 비교표 또는 흐름

| 개념 | 의미 |
|---|---|
| 블랙박스 모델 | 내부 판단 근거를 이해하기 어려운 모델 |
| 설명가능성 기법 | 판단 근거를 시각화·제시하는 방법 |
| 활용 | 신뢰성 확보, 편향 발견에 기여 |

## 실무 사례

한 병원이 폐렴 진단 영상 AI의 판단 근거를 시각화했더니, 모델이 실제 병변이 아닌 영상 가장자리의 촬영 장비 표식을 근거로 판단하고 있음을 발견해 모델을 재학습한 사례가 있다.

## 국가시험 포인트

- 설명가능성 개념과 블랙박스 모델의 한계를 확인하는 문제

## 자주 하는 실수

- 성능이 높은 모델은 설명가능성이 없어도 무조건 신뢰할 수 있다고 오해하는 것

## 관련 법령·표준

이 Topic은 개념 설명으로 특정 법령에 근거하지 않는다.

## AI Note

설명가능성은 AI 결과를 비판적으로 검토하고 편향 등 숨겨진 문제를 발견하는 데 핵심적인 도구다.

## 핵심 요약

- 설명가능성은 AI의 판단 근거를 사람이 이해할 수 있게 제시하는 능력이다.
- 블랙박스 모델의 한계를 보완하고 편향 발견에도 기여한다.

## 플래시카드

1. Q. 내부 작동 방식을 사람이 이해하기 어려운 복잡한 모델을 무엇이라 하는가? / A. 블랙박스(Black Box) 모델
2. Q. 모델의 판단 근거를 사람이 이해할 수 있게 제시하는 능력은? / A. 설명가능성(Explainability)
