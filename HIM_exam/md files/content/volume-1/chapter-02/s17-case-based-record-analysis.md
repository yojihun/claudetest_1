# 사례 기반 기록 분석

## 학습목표

1. Chapter 2에서 다룬 기록 구성·작성 원칙·완결성 개념을 하나의 사례에 통합 적용할 수 있다.
2. 정량분석·정성분석·미비기록 관리·법적 가치 판단을 순서대로 수행할 수 있다.

## 한 문장 정의

## 핵심 개념

사례 분석의 표준 절차:

1. 정량분석: 필수 문서·서명·날짜가 모두 있는가
2. 정성분석: 기록 간 내용이 논리적으로 일치하는가
3. 완전성 판단: 문서 단위·전체 단위·내용적 완전성 중 무엇이 문제인가
4. 미비기록 처리: 무엇을, 누구에게, 어떻게 보완 요청할 것인가
5. 법적 가치 검토: 이 상태로 분쟁이 발생하면 어떤 위험이 있는가

## 상세 설명

아래 사례를 위 다섯 단계로 분석해본다.

> **사례:** A환자는 담낭절제술을 받고 퇴원했다. 기록을 검토한 결과, 수술 동의서는 담낭절제술로 기재되어 있고 수술기록의 수술명도 동일하다. 그러나 마취기록에는 서명이 없고, 퇴원요약은 퇴원 후 3주가 지나서야 작성되었으며, 퇴원요약에는 "특이 합병증 없음"이라고만 적혀 있는데 경과기록에는 수술 후 발열이 3일간 지속되었다는 기록이 있다.

## 비교표 또는 흐름

| 단계 | 이 사례에서의 판단 |
|---|---|
| 정량분석 | 마취기록 서명 누락 발견 |
| 정성분석 | 퇴원요약("특이 합병증 없음")과 경과기록(발열 지속)의 모순 발견 |
| 완전성 판단 | 문서 단위(서명 누락) + 내용적 완전성(모순된 요약) 문제가 동시에 존재 |
| 미비기록 처리 | 마취과의에게 서명 요청, 담당의에게 퇴원요약 내용 수정 요청 |
| 법적 가치 검토 | 퇴원요약 지연과 모순된 내용은 향후 분쟁 시 병원에 불리하게 작용할 수 있음 |

## 국가시험 포인트

- 하나의 사례에서 여러 문제(서명 누락, 완결성, 모순, 지연)를 동시에 찾아내는 능력
- 각 문제를 어떤 Topic의 개념으로 설명할 수 있는지 연결하는 능력

## 자주 하는 실수

- 사례에서 한 가지 문제만 찾고 나머지를 놓치는 것
- 정량분석에서 발견한 문제(서명 누락)와 정성분석에서 발견한 문제(내용 모순)를 구분하지 않고 뭉뚱그려 설명하는 것

## 관련 법령·표준

이 사례의 법적 가치 판단은 [기록의 법적 가치](s16-legal-value-of-records.md)의 관련 법령 항목을 참고한다.

## 핵심 요약

- 사례 분석은 정량분석 → 정성분석 → 완전성 판단 → 미비기록 처리 → 법적 가치 검토의 순서로 진행한다.
- 하나의 사례에 여러 층위의 문제가 동시에 존재할 수 있다.
