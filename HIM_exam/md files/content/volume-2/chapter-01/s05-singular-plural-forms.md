# 단수와 복수형

## 학습목표

1. 그리스어·라틴어 기원 의학용어의 불규칙 복수형 규칙을 설명할 수 있다.
2. 대표적인 단수-복수형 변환 예를 들 수 있다.

## 한 문장 정의

의학용어의 상당수는 그리스어·라틴어에서 유래해, 영어의 일반적인 "-s"를 붙이는 규칙이 아닌 고유한 복수형 변환 규칙을 따른다.

## 왜 중요한가

복수형 규칙을 모르면 의무기록이나 검사 결과지에서 단수·복수 표기를 혼동해 정보를 잘못 이해하거나 잘못 기록할 수 있다.

## 선수 개념

- [결합모음](s04-combining-vowels.md)

## 핵심 개념

- -a → -ae (예: Vertebra → Vertebrae)
- -is → -es (예: Diagnosis → Diagnoses)
- -um → -a (예: Bacterium → Bacteria)
- -us → -i (예: Bronchus → Bronchi, 단 일부 예외 존재)
- -on → -a (예: Ganglion → Ganglia)

## 상세 설명

이 규칙들은 영어의 일반적 복수형("-s"를 붙이는 것)과 다르므로 별도로 암기해야 한다. 예를 들어 "Diagnosis"의 복수형은 "Diagnosises"가 아니라 "Diagnoses"이며, "Vertebra"의 복수형은 "Vertebras"가 아니라 "Vertebrae"이다. 이런 불규칙 변환은 의무기록에서 여러 개의 진단명(Diagnoses), 여러 개의 척추뼈(Vertebrae)를 언급할 때 정확한 철자로 나타난다.

## 비교표 또는 흐름

| 단수 | 복수 | 규칙 |
|---|---|---|
| Vertebra | Vertebrae | -a → -ae |
| Diagnosis | Diagnoses | -is → -es |
| Bacterium | Bacteria | -um → -a |
| Bronchus | Bronchi | -us → -i |
| Ganglion | Ganglia | -on → -a |

## 실무 사례

퇴원요약지에 "Multiple diagnoses were confirmed"라는 문장이 있을 때, Diagnoses가 Diagnosis의 복수형임을 알아야 "여러 개의 진단이 확인되었다"는 의미를 정확히 이해할 수 있다.

## 국가시험 포인트

- 불규칙 복수형 변환 규칙(특히 -is→-es, -um→-a)을 확인하는 문제

## 자주 하는 실수

- 의학용어에도 영어의 일반 규칙("-s"만 붙이기)을 그대로 적용하는 것

## 관련 법령·표준

## 핵심 요약

- 의학용어의 복수형은 그리스어·라틴어 기원의 고유한 규칙을 따른다.
- -a→-ae, -is→-es, -um→-a, -us→-i, -on→-a 등의 변환 규칙을 암기해야 한다.
