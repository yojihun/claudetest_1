# 표기 원칙

- 첫 등장: 한국어(영어, 약어)
- 이후: 문맥에 따라 한국어 또는 널리 쓰이는 약어
- 약어만으로 시작하지 않는다.
- 동일 영문 용어에 여러 한국어 번역이 있으면 프로젝트 표준어를 우선한다.
- 법령상 명칭이 일반 번역과 다르면 법령상 명칭을 사용한다.

| 처방전달시스템 | Order Communication System | OCS | 처방과 관련 정보를 부서에 전달하는 시스템 |
| 의료영상저장전송시스템 | Picture Archiving and Communication System | PACS | 의료영상 저장·조회·전송 시스템 |
| 검사실정보시스템 | Laboratory Information System | LIS | 검사 업무와 결과를 관리하는 시스템 |

| 객관적 정보 | Objective | O | 관찰·검사로 확인된 정보 |
| 평가 | Assessment | A | 수집된 정보에 대한 임상적 평가 |
| 계획 | Plan | P | 진단·치료·추적 계획 |
| 정확성 | Accuracy | - | 기록이 실제 사실과 일치하는 정도 |
| 완전성 | Completeness | - | 필요한 정보가 누락되지 않은 정도 |
| 적시성 | Timeliness | - | 필요한 시점에 기록·제공되는 정도 |
| 일관성 | Consistency | - | 자료 간 정의와 값이 모순되지 않는 정도 |
| 무결성 | Integrity | - | 정보가 허가 없이 변경·훼손되지 않은 상태 |
| 프라이버시 | Privacy | - | 개인이 자신의 사생활과 정보 이용을 통제할 권리 |
| 비밀보장 | Confidentiality | - | 알게 된 정보를 허가 없이 공개하지 않을 의무 |
| 보안 | Security | - | 정보를 보호하기 위한 기술적·관리적 조치 |
| 책임성 | Accountability | - | 접근과 행위의 책임 주체를 추적할 수 있는 성질 |
| 상호운용성 | Interoperability | - | 서로 다른 시스템이 정보를 교환하고 의미 있게 사용하는 능력 |

| 주진단 | Principal Diagnosis | - | 입원 후 검사 결과 입원의 주된 원인으로 확정된 진단; 적용 기준 확인 필요 |
| 기타진단 | Other Diagnosis | - | 주진단 외 환자 진료에 영향을 준 진단 |

| 암등록 | Cancer Registration | - | 암 발생과 진료 정보를 일정 기준에 따라 수집·관리하는 활동 |
| 원발부위 | Primary Site | - | 암이 처음 발생한 해부학적 부위 |

| 병상회전율 | Bed Turnover Rate | - | 일정 기간 병상 한 개당 퇴원환자 수 |

# 관리 필요 용어

다음 용어는 공식 번역과 최신 기준을 확인한 후 확정한다.

- 보건의료정보관리사의 공식 영문 명칭
- 의료행위 분류체계의 공식 명칭과 적용 버전
- 임상문서개선의 국내 공식 사용 용어
- 가명정보·비식별정보 관련 법적 구분
- 상급종합병원 및 의료기관 종별 최신 법정 기준
