# 표기 원칙

- 첫 등장: 한국어(영어, 약어)
- 이후: 문맥에 따라 한국어 또는 널리 쓰이는 약어
- 약어만으로 시작하지 않는다.
- 동일 영문 용어에 여러 한국어 번역이 있으면 프로젝트 표준어를 우선한다.
- 법령상 명칭이 일반 번역과 다르면 법령상 명칭을 사용한다.

# 의학용어 (2026 보건의료정보관리사 국가시험 기출범위 반영)

각 계통별로 흩어져 있는 용어들을 앱이나 데이터베이스에서 파싱하기 쉽도록 1:1 (용어:의미) 관계로 엄격하게 재구성한 표입니다.

# 핵심 용어 (보건의료정보학)

| 용어 (Term) | 의미 (Meaning) |
|---|---|
| Health Information Management | 보건의료정보를 수집·관리·분석·보호·활용하는 전문 분야 |
| Health Information Manager | 국내 법정 명칭과 영문 표기는 공식 자료 확인 필요 |
| Medical Record | 환자의 진료와 관련하여 작성된 기록 (의료기록, 의무기록) |
| Electronic Medical Record | 한 의료기관 내부의 전자 진료기록 |
| Electronic Health Record | 기관 간 연속성과 공유를 지향하는 건강기록 |
| Personal Health Record | 개인이 접근·관리하는 건강정보 |
| Hospital Information System | 병원 업무를 지원하는 통합 정보시스템 |
| Order Communication System | 처방과 관련 정보를 부서에 전달하는 시스템 |
| Picture Archiving and Communication System | 의료영상 저장·조회·전송 시스템 |
| Laboratory Information System | 검사 업무와 결과를 관리하는 시스템 |
| Clinical Data Warehouse | 분석을 위해 임상데이터를 통합한 저장소 |
| Problem-Oriented Medical Record | 문제목록 중심으로 구성된 기록 방식 |
| Subjective | 환자가 호소하거나 진술한 정보 |
| Objective | 관찰·검사로 확인된 정보 |
| Assessment | 수집된 정보에 대한 임상적 평가 |
| Plan | 진단·치료·추적 계획 |
| Accuracy | 기록이 실제 사실과 일치하는 정도 |
